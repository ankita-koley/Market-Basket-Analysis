## Market Basket Analysis

## Project Overview
This project performs Market Basket Analysis on grocery transaction data using the Apriori algorithm to identify frequently purchased product combinations. It generates association rules (Support, Confidence, and Lift) and visualizes insights through an interactive Power BI dashboard to support cross-selling decisions.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Power BI

## Project Workflow
- Data Cleaning
- Transaction Preparation
- Apriori Algorithm
- Association Rule Mining
- Data Visualization
-Power BI Dashboard
